- To display only directories in a folder
	ls -d */
	or 
	ls -ld */ to display as a list

- To use VIM to edit a text file
	vim practice.txt
	i → write three lines → Esc
	dd to delete a line, u to undo
	/line to search
	:wq

	:w
	Save
	:q
	Quit (fails if unsaved changes)
	:wq
	Save and quit
	:q!
	Quit without saving
	:x
	Save and quit (like :wq if changed)
